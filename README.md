## Getting Started


### Prerequisites

Required framework :
* **ROS Kinetic**

Required Hardware :
* **Raspberry Pi 3**
* **Arduino MEGA 2560**
* **XBOX 360 Kinect**

Required ROS packages :
* **freenect_stack**
* **rosserial-arduino**

Required libraries :
* **OpenCV 3.4.0**+

### Installing

Clone the repository
```
git clone https://github.com/raymas/Rover.git
```
